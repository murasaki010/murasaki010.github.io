# 道路同伦

道路同伦是相对同伦的特例，继承相关性质。

Definition. (X,τ)(X,\tau)(X,τ) 中的道路 (path) 是指连续映射 f:I→Xf:I\to Xf:I→X。

Remark. 注意区分道路和道路像，这要求我们引出道路同伦类。

Definition. (X,τ)(X,\tau)(X,τ) 中的 （道路）同伦 (homotopy) 指的是一族映射 {ft}t∈[0,1]\{f_t\}_{t\in[0,1]}{ft​}t∈[0,1]​，满足

Remark. 道路同伦描述了固定始终点，道路 f0f_0f0​ 连续 “形变” 到道路 f1f_1f1​ 的过程。

Remark. 同伦映射 FFF 在正方形定义域上是连续的。

# 回路与基本群

将始终点合为一点，则就有回路。回路是特殊的道路。回路的一些性质也可以推广到道路，但是并不重要。

Definition. 回路 (loop) 是到 (X,τ)(X,\tau)(X,τ) 的连续映射，满足

α:I→X,α(0)=α(1)\alpha:I\to X,\quad \alpha(0)=\alpha(1)α:I→X,α(0)=α(1)

称 α(0)\alpha(0)α(0) 是回路的基点 (base point)。

Remark. 和一般道路一样，回路只要给定基点，就确定了同基点的同伦类，回路 α\alphaα 对应的同伦类记为 [α][\alpha][α]，以 x0x_0x0​ 为基点的同伦类全体构成 π1(X,x0)\pi_1(X,x_0)π1​(X,x0​)，即基本群 (fundamental group) 或一维同伦群。

Theorem. π1(X,x0)\pi_1(X,x_0)π1​(X,x0​) 在同伦类乘法 [α.β]=[α]⋅[β][\alpha.\beta]=[\alpha]\cdot [\beta][α.β]=[α]⋅[β] 下构成群。

Theorem. 道路连通空间中不同基点的基本群之间同构。

# 同伦等价

同伦等价的定义比同胚更弱，但是在同伦等价的意义下，许多同胚不变量也得以保持。

Definition. 拓扑空间 X,YX,YX,Y 有相同的同伦型 (homotopy type)，或者说是同伦等价 (homotopo equivalent) 的，记为 X≃YX\simeq YX≃Y，如果存在连续映射满足

f:X→Y,g:Y→X:g∘f≃1X,f∘g≃1Yf:X\to Y,g:Y\to X:\quad g\circ f\simeq 1_X,\quad f\circ g\simeq 1_Yf:X→Y,g:Y→X:g∘f≃1X​,f∘g≃1Y​

称 f,gf,gf,g 互为同伦逆 (homotopy inverse)。

Remark. 同伦等价将复合为单位弱化为复合同伦于单位。

Remark. 同理可以定义相对同伦等价。

Lemma. 同伦等价是拓扑空间的等价关系。

Corollary. 同胚蕴含同伦等价。

# 形变收缩

形变收缩是单位映射的相对同伦。

Definition. 拓扑空间 XXX 及其子空间 AAA，称相对 AAA 的同伦映射 G:X×I→XG:X\times I\to XG:X×I→X 是 XXX 到 AAA 的形变收缩 (deformation retraction)，如果满足

G(x,0)=x,G(x,1)∈A,∀x∈X;G(x,t)=x,∀t∈I,x∈AG(x,0)=x,G(x,1)\in A,\forall x\in X;\quad G(x,t)=x,\forall t\in I,x\in AG(x,0)=x,G(x,1)∈A,∀x∈X;G(x,t)=x,∀t∈I,x∈A

称 A⊆XA\subseteq XA⊆X 是 GGG 的形变收缩子空间 (deformation retracted subspace)。

Remark. 一般而言，收缩 (retraction) 指的是 r:X→Xr:X\to Xr:X→X 满足 r2=rr^2=rr2=r 的连续映射，它是拓扑意义下的投影算子。形变收缩映射 GGG 中的 G(x,1)∈AG(x,1)\in AG(x,1)∈A 为 XXX 到 AAA 的收缩。形变收缩诱导收缩，但是收缩不一定对应存在一个形变收缩。

Remark. 如果存在 XXX 到 AAA 的形变收缩，则 X≃AX\simeq AX≃A。考虑嵌入映射。

Corollary. 形变收缩到同一个子空间的拓扑空间是同伦等价的。

这要求 π(x)=π(x′)\pi(x)=\pi(x')π(x)=π(x′) 的点都满足

π∘F~(x,t)=π∘F~(x′,t),∀t∈I\pi \circ \tilde F(x,t)=\pi \circ \tilde F(x',t),\quad \forall t\in Iπ∘F~(x,t)=π∘F~(x′,t),∀t∈I

# 可缩空间

Definition. 拓扑空间 XXX 是可缩的 (contractible)，如果 XXX 同伦等价于单点空间。

Definition. 连续映射称为零伦 (nullhomotopic)，如果其同伦于常值映射。

Corollary. 可缩空间是单连通空间。

Theorem. 可缩空间当且仅当恒等映射零伦。

Lemma. 像空间是可缩空间的连续映射是同伦的。

Corollary. 可缩空间 XXX 的恒等映射同伦于任意常值映射 cxc_xcx​，其中 x∈Xx\in Xx∈X。

Example. 锥 CXCXCX 是可缩空间。凸空间是可缩空间。

Remark. 形变收缩到单点集蕴含可缩。前者还多了保持形变收缩子空间不变的性质。

Example. 梳子空间可以形变收缩到单点集，但不可缩。梳子空间定义为

X=([0,1]×{0})∪(⋃n=1∞{1n}×[0,1])∪({0}×[0,1])⊆R2X=([0,1]\times\{0\})\cup \left(\bigcup_{n=1}^\infty \left\{\tfrac 1 n\right\}\times [0,1]\right)\cup (\{0\}\times [0,1])\subseteq \mathbb R^2X=([0,1]×{0})∪(n=1⋃∞​{n1​}×[0,1])∪({0}×[0,1])⊆R2
