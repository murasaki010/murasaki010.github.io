# 连通性

有两种观点看待 “连通性”：

（1）集合论观点：XXX 是连通的，当且仅当 X=A∪BX=A\cup BX=A∪B，其中 A,B≠∅A,B\neq \emptysetA,B=∅ 满足 A‾∩B≠∅\overline A\cap B\neq\emptysetA∩B=∅ 或 B‾∩A≠∅\overline B\cap A\neq\emptysetB∩A=∅.

（2）几何观点：记 I=[a,b]⊆RI=[a,b]\subseteq \mathbb RI=[a,b]⊆R，XXX 是连通的，当且仅当对于任意 p,q∈Xp,q\in Xp,q∈X，存在 f:I→Xf:I\to Xf:I→X 是连续映射，使得 f(0)=p,f(1)=qf(0)=p,f(1)=qf(0)=p,f(1)=q.

我们将第一个观点的概念称为连通性，第二个观点的概念称为道路连通性。事实上，道路连通蕴含连通，反过来则不一定成立。

Definition. 空间 XXX 称作连通的，如果对任意 A,B≠∅A,B\neq \emptysetA,B=∅ 满足 X=A∪BX=A\cup BX=A∪B，总是有 A‾∩B≠∅\overline A\cap B\neq \emptysetA∩B=∅ 或 B‾∩A≠∅\overline B\cap A\neq \emptysetB∩A=∅.
