# Convex polyhedral cones

本节收录了关于凸多面锥的基本事实，这些内容将是后续所需的。这些结果在其自然的一般性框架下可以在许多关于凸几何或代数几何的教材中找到。然而，由于这些结果相对简单，自己去证明它们几乎和引用文献一样容易。我们之所以也附上证明，是因为它们展示了如何找到半群 [半群是一个配备二元运算的非空集合，该运算满足结合律] 的生成元，而这正是我们进行具体计算时所需要的.

设 VVV 为向量空间 NRN_{\mathbb R}NR​，其对偶空间为 V∗=MRV^*=M_{\mathbb R}V∗=MR​. 一个凸多面锥 (convex polyhedral cone) 是如下形式的集合：

σ={r1v1+⋯+rkvk∈V∣ri≥0}\sigma =\{r_1v_1+\cdots +r_kv_k\in V|r_i\geq 0\}σ={r1​v1​+⋯+rk​vk​∈V∣ri​≥0}

它由任意有限个向量 v1,⋯,vk∈Vv_1,\cdots,v_k\in Vv1​,⋯,vk​∈V 的线性组合生成。这些向量 —— 有时也包括它们对应的射线 —— 即由某个 viv_ivi​ 的正数倍数组成的集合 {rvi∣r>0}\{r v_i|r>0\}{rvi​∣r>0}—— 被称为锥 σ\sigmaσ 的生成元.

[对偶空间是由所有线性函数 V→RV\to\mathbb RV→R 组成的向量空间；特别指出组合是非负线性组合，如图；之后的线性组合一般都是非负线性组合，除非写成 σ+(−σ)\sigma+(-\sigma)σ+(−σ) 这种形式]

我们很快将看到锥的对偶描述，即作为半空间的交集。锥 (\sigma) 的维数 (\dim (\sigma) ) 是指由 ( \sigma ) 张成的线性空间 ( \mathbbR} \sigma =\sigma+(-\sigma)\) 的维数。任意集合 \( \sigma \) 的 ** 对偶锥 ** \( \sigma(\vee \) 是支撑超平面方程的集合，即： \[ \sigma^\vee = \{ u \in V^* \mid \langle u, v \rangle \geq 0 \text { for all)v \in \sigma }
]

所有这些都基于凸集理论中的以下基本事实：

  1. 如果 (\sigma) 是一个凸多面锥，且 ( v_0 \notin \sigma )，那么存在某个 ( u_0 \in \sigma^\vee )，使得 ( \langle u_0, v_0 \rangle < 0 ).

[参见凸集书籍，这里认同即可]

我们列出由 (1) 推出的一些结论。对 (1) 的直接转化就是对偶定理：

  1. ( (\sigma\vee)\vee = \sigma ).

[只需证明 ⊂\subset⊂ 方向，这用到了 (1) ]

面（face）(\tau) 是 (\sigma) 与某个支撑超平面的交集：

[
\tau = \sigma \cap u^\perp = { v \in \sigma \mid \langle u, v \rangle = 0 }, \quad \text{for some } u \in \sigma^\vee.
]

锥本身也被视为自己的一个面，而其他面称为真面（proper faces）。注意，锥的任何线性子空间都包含在该锥的每一个面中.

[考虑零算子，则锥本身就是平凡的面；线性子空间是双向的，但锥是单向的，需要注意这一点，不过命题根据面的定义和线性就可以得到]

  1. 任意面也是一个凸多面锥.

面 (\sigma \cap u^\perp) 由锥 (\sigma) 的生成集里满足 (\langle u, v_i \rangle = 0) 的那些向量 (v_i) 生成。特别地，我们可以看出一个锥只有有限多个面.

[这是几何意义，面就是锥的一部分棱张成的锥，具体而言，σ∨\sigma^\veeσ∨ 中的算子和生成元满足不等关系，而面则是将部分不等式改成等式，从而得到的子锥；注意到锥内部都是棱（生成元）的非负线性组合，所以如果内部达到零，则对应的生成元必须为零，这样一个锥只有有限个面]

  1. 任意多个面的交集也是一个面.

这可由如下等式看出：(
\bigcap (\sigma \cap u_i^\perp) = \sigma \cap \left ( \sum u_i \right)^\perp)，其中 ui∈σ∨u_i \in \sigma^\veeui​∈σ∨.

  1. 面的面也是一个面.

事实上，若 (\tau = \sigma \cap u^\perp)，且 (\gamma = \tau \cap (u')^\perp)，其中 (u \in \sigma^\vee)，(u' \in \tau^\vee)，则对于足够大的正数 (p)，有 (u' + p u \in \sigma^\vee)，且 (\gamma = \sigma \cap (u' + p u)^\perp.)

[根据 (3) u′u'u′ 的支撑集由 σ\sigmaσ 的生成元中满足 ⟨u′,vi⟩=0\langle u',v_i\rangle=0⟨u′,vi​⟩=0 的那些向量张成，所以对于足够大的 ppp，u′+puu'+puu′+pu 也在对偶锥中]
