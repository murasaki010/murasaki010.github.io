# 自反 Banach 空间

我们还可以进一步假设 lll 是满射，从而得到自反的概念。

Definition. l:X→X∗∗l:X\to X^{**}l:X→X∗∗ 是等距同构时，称 XXX 是自反 (reflexive) 的。

Remark. 定义中的 lll 特指规范嵌入。其他的等距同构不是自反的考虑对象。

Remark. 对偶空间 X∗∗X^{**}X∗∗ 总是 Banach 空间，因此自反空间一定是 Banach 空间。

Remark. 证明自反，只需证明 lll 是满射。
