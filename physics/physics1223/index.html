# 黑体辐射

热辐射： 能量按频率的分布随温度而不同的电磁辐射叫做热辐射。

平衡热辐射： 如果在同一时间内从物体表面辐射的电磁波能量和它吸收的电磁波能量相等，物体和辐射就处于温度一定的热平衡状态，此时的热辐射称为平衡热辐射。

以下的讨论都是基于平衡热辐射。

光谱辐射出射度： 频率 ν\nuν 的光谱辐射出射度是指单位时间内从物体单位表面积发出的频率在 ν\nuν 附近单位频率区间的电磁波的能量。光谱辐射出射度（按频率分布）用 MνM_\nuMν​ 表示。设单位时间从单位表面（温度 TTT ）发出的频率在 ν\nuν 附近的电磁波能量为 dEν(T)\mathrm dE_\nu(T)dEν​(T)，则

# 光电效应

光电效应： 光照射某些金属时，能从表面释放出电子的效应。光电效应中产生的电子称为光电子，在电场加速下形成光电流。记光强 III。有以下实验规律

  1. 存在饱和光电流 imi_mim​：在 III 一定时，增大加速电压，光电流能达到最大值 imi_mim​
  2. 在 ν\nuν 一定时，im∝Ii_m\propto Iim​∝I，这也说明单位时间逸出的光电子数与光强成正比
  3. 存在截止电压 Uc=Kν−U0U_c=K\nu-U_0Uc​=Kν−U0​，与 III 无关
  4. 存在红限波长与红限频率 ν0=U0K\nu_0=\dfrac{U_0}Kν0​=KU0​​
  5. 最大初动能 Ek,max=12mvm2=UcE_{k,\mathrm{max}}=\dfrac 12mv_m^2=U_cEk,max​=21​mvm2​=Uc​
  6. 光电子的逸出几乎是在光照到金属表面上的同时发生的，延迟时间在 10−9s10^{-9}s10−9s 以下

补充，考虑到原子质量较大，光电效应中电子 - 光子系统动量也近似于守恒。

# 康普顿散射

康普顿散射： X 射线通过物质时向各方向散射，其中有波长改变的散射称为康普顿散射。散射曲线有以下特点

  1. 除了原波长 λ0\lambda_0λ0​ 外，出现了新的散射波长 λ\lambdaλ
  2. 新波长 λ\lambdaλ 随散射角 φ\varphiφ 的增大而增大
  3. 当 φ\varphiφ 增大时，原波长的谱线强度降低，而新波长的谱线强度升高

康普顿散射公式： X 射线的散射是单个光子和单个电子发生弹性碰撞的结果，其中电子在碰撞前视作静止的。利用能量守恒和动量守恒

Δλ=λ−λ0=hm0c(1−cos⁡φ)=λC(1−cos⁡φ)\Delta\lambda=\lambda-\lambda_0=\dfrac h{m_0c}(1-\cos \varphi)=\lambda_C (1-\cos \varphi)Δλ=λ−λ0​=m0​ch​(1−cosφ)=λC​(1−cosφ)

其中 λC\lambda_CλC​ 称为康普顿波长。散射波长与散射物质无关。

此外还有一些补充：

  1. 瑞利散射：波长不变的散射。因为光子还可以与原子核中束缚得很紧的内层电子发生弹性碰撞
  2. 只有当入射波波长与电子的康普顿波长相当时，才能显著观测到
  3. 自由电子不能吸收光子，只能散射光子
