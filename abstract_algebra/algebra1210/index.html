# Galois 对应

定义 给定正规扩张 L/K^L/_KL/K​，记 Gal(L/K)=autK(L)\mathrm{Gal}({^L/_K})=\mathrm{aut}_K(L)Gal(L/K​)=autK​(L) 并称之为 L/K^L/_KL/K​ 的 Galois 群。如果域扩张 L/K^L/_KL/K​ 是正规的也是可分的，就称 L/K^L/_KL/K​ 是 Galois 扩张。对任意的子群 H<Gal(L/K)H<\mathrm{Gal}(^L/_K)H<Gal(L/K​)，定义

LH:={x∈L:g(x)=x,∀g∈H}L^H:=\{x\in L:g(x)=x,\forall g\in H\}LH:={x∈L:g(x)=x,∀g∈H}

引理 L/K^L/_KL/K​ 是有限正规扩张，则

[L:K]s=∣Gal(L/K)∣[L:K]_s=|\mathrm{Gal}(^L/_K)|[L:K]s​=∣Gal(L/K​)∣

特别地，若 L/K^L/_KL/K​ 是 Galois 扩张，那么

[L:K]=∣Gal(L/K)∣[L:K]=|\mathrm{Gal}(^L/_K)|[L:K]=∣Gal(L/K​)∣

命题 L/K^L/_KL/K​ 是正规扩张，G=Gal(L/K)G=\mathrm{Gal}({^L/_K})G=Gal(L/K​)。那么 LG/K^{L^G}/_KLG/K​ 是纯不可分的， L/LG^L/_{L^G}L/LG​ 是 Galois 扩张并且

Gal(L/LG)=G\mathrm{Gal}(^L/_{L^G})=GGal(L/LG​)=G

进一步，有 L‾s⋅LG=L\overline L^s\cdot L^G=LLs⋅LG=L，L‾s∩LG=K\overline L^s\cap L^G=KLs∩LG=K

Remark

GGG 在根上的作用是 Galois 理论中最典型的推理方法（群作用）

定理（Galois 对应定理）L/K^L/_KL/K​ 是有限 Galois 扩张，定义中间域的集合和子群的集合：

M={K⊂M⊂L:M是中间域}\mathcal M=\{K\subset M\subset L:M\text{ 是中间域}\}M={K⊂M⊂L:M 是中间域}

S={H<Gal(L/K)是子群}\mathcal S=\{H<\mathrm{Gal}({^L/_K})\text{ 是子群}\}S={H<Gal(L/K​) 是子群}

并用包含关系作为 M\mathcal MM 和 S\mathcal SS 上的偏序。那么，有如下反转偏序关系的 Galois 对应（以下映射互为逆）

M→1:1S,M→Gal(L/M),LH←H\mathcal M\xrightarrow{1:1} S,\quad M\to \mathrm {Gal}(^L/_M),\ \ L^H\leftarrow HM1:1​S,M→Gal(L/M​), LH←H

进一步，扩张 M/K^M/_KM/K​ 是正规扩张当且仅当 Gal(L/M)◃Gal(L/K)\mathrm{Gal}({^L/_M})\triangleleft \mathrm{Gal}({^L/_K})Gal(L/M​)◃Gal(L/K​) 是正规子群，在此情形下

Gal(M/K)=Gal(L/K)/Gal(L/M)\mathrm {Gal}({^M/_K})={^{\mathrm{Gal}({^L/_K})}/_{\mathrm{Gal}({^L/_M})}}Gal(M/K​)=Gal(L/K​)/Gal(L/M​)​

Remark

其中需要证明 LGal(L/M)=ML^{\mathrm{Gal}({^L/_M})}=MLGal(L/M​)=M 的结论。这不要求 L/K^L/_KL/K​ 是有限扩张。作为应用：对任意的 α∈Q‾\alpha\in \overline{\mathbb Q}α∈Q​，则 α∈Q\alpha\in \mathbb Qα∈Q 当且仅当对任意的 σ∈Gal(Q‾/Q)\sigma\in\mathrm{Gal}({^{\overline{\mathbb Q}}/_{\mathbb Q}})σ∈Gal(Q​/Q​) 都有 σ(α)=α\sigma(\alpha)=\alphaσ(α)=α

命题 如果 L/K^L/_KL/K​ 是有限正规扩张，则

[L:LG]=[L:K]s=Gal(L/K),[LG:K]=[L:K][L:K]s=L/K的不可分次数[L:L^G]=[L:K]_s=\mathrm{Gal}({^L/_{K}}),\quad [L^G:K]=\dfrac{[L:K]}{[L:K]_s}={^L/_K}\text{ 的不可分次数}[L:LG]=[L:K]s​=Gal(L/K​),[LG:K]=[L:K]s​[L:K]​=L/K​ 的不可分次数

命题（Galois 下降）给定有限维 Galois 扩张 L/K^L/_KL/K​ 和 nnn - 维 LLL - 线性空间 LnL^nLn，其中 e1,...,ene_1,...,e_ne1​,...,en​ 是标准基。对任意的 x∈(x1,...,xn)∈Lnx\in (x_1,...,x_n)\in L^nx∈(x1​,...,xn​)∈Ln 和 σ∈Gal(L/K)\sigma\in\mathrm {Gal}({^L/_K})σ∈Gal(L/K​)，定义

σ(x)=(σ(x1),σ(x2),...,σ(xn))\sigma(x)=(\sigma(x_1),\sigma(x_2),...,\sigma(x_n))σ(x)=(σ(x1​),σ(x2​),...,σ(xn​))

假设 V⊂LnV\subset L^nV⊂Ln 是 VVV 的 ddd - 维 LLL - 线性子空间并且在 Gal(L/K)\mathrm {Gal}({^L/_K})Gal(L/K​) 作用下不变。那么，存在 VVV 的基 v1=(v11,...,v1n),...,vdv_1=(v_{11},...,v_{1n}),...,v_dv1​=(v11​,...,v1n​),...,vd​，使得 vij∈Kv_{ij}\in Kvij​∈K。

# Galois 群在根上的作用

定理 KKK 是域，P∈K[X]P\in K[X]P∈K[X] 为可分的多项式，LLL 是 PPP 的分裂域。那么 Gal(L/K)\mathrm{Gal}({^L/_K})Gal(L/K​) 在根的集合上的作用 ZP(L)\mathrm Z_P(L)ZP​(L) 是忠实的

1→Gal(L/K)→SZP(L)1\to \mathrm{Gal}({^L/_K})\to\mathfrak S_{\mathrm Z_P(L)}1→Gal(L/K​)→SZP​(L)​

这个作用是传递的当且仅当 PPP 是不可约多项式。

Remark

假设 LLL 是可分多项式 P(X)∈K[X]P(X)\in K[X]P(X)∈K[X] 的分裂域并且 PPP 是 mmm 个不可约多项式的乘积

P(X)=P1(X)⋯Pm(X)P(X)=P_1(X)\cdots P_m(X)P(X)=P1​(X)⋯Pm​(X)

那么，Gal(L/K)↷ZP(L)^{\mathrm{Gal}({^L/_K})↷}\mathrm Z_P(L)Gal(L/K​)↷ZP​(L) 具有 mmm 个轨道，并且每个轨道都对应着某个 PiP_iPi​ 的根。
