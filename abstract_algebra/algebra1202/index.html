# 分裂域

定义 4.2.1.1 KKK 是域，{Pi(X)}i∈I\{P_i(X)\}_{i\in I}{Pi​(X)}i∈I​ 是 K[X]K[X]K[X] 中一族多项式，L/K^L/_KL/K​ 是域扩张。如果

  1. 所有多项式 {Pi(X)}i∈I\{P_i(X)\}_{i\in I}{Pi​(X)}i∈I​ 在 LLL 中分裂，即 ∀i∈I\forall i\in I∀i∈I，存在 αi,j\alpha_{i,j}αi,j​ 以及 ai∈Ka_i\in Kai​∈K，使得

Pi(X)=ai(X−αi,1)(X−αi,2)⋯(X−αi,ni)P_i(X)=a_i(X-\alpha_{i,1})(X-\alpha_{i,2})\cdots (X-\alpha_{i,n_i})Pi​(X)=ai​(X−αi,1​)(X−αi,2​)⋯(X−αi,ni​​)

  1. L=K({αi,j}i∈I,j≤ni)L=K(\{\alpha_{i,j}\}_{i\in I,j\leq n_i})L=K({αi,j​}i∈I,j≤ni​​)

则称 LLL 是 KKK 的（由 {Pi}i∈I\{P_i\}_{i\in I}{Pi​}i∈I​ 给出的）一个 分裂域。

Remark

分裂域 LLL 是 KKK 的代数扩张。KKK 的分裂域的构造就是把一些多项式的所有根都加到 KKK 中。

命题 4.2.1.2（分裂域的下降、唯一性 ） KKK 是域，{Pi(X)}i∈I\{P_i(X)\}_{i\in I}{Pi​(X)}i∈I​ 是 K[X]K[X]K[X] 中一族多项式，LLL 和 L′L'L′ 均为 KKK 的由 {Pi}i∈I\{P_i\}_{i\in I}{Pi​}i∈I​ 定义的分裂域，并且 L′L'L′ 在某个代数封闭域 Ω\OmegaΩ 中。那么

hom⁡K(L,Ω)⊂hom⁡K(L,L′)\hom_K(L,\Omega)\subset \hom_K(L,L')homK​(L,Ω)⊂homK​(L,L′)


特别地，分裂域在 KKK - 同构意义下唯一。

# 正规性

定理 4.2.2.1（正规性的定义） 给定代数扩张 L/K^L/_KL/K​，下面叙述等价：

  1. LLL 是 K[X]K[X]K[X] 中一族多项式 {Pi}i∈I\{P_i\}_{i\in I}{Pi​}i∈I​ 的分裂域；
  2. 对任意域扩张 Ω/L^\Omega/_LΩ/L​，其中 Ω\OmegaΩ 是代数封闭域，成立 hom⁡K(L,Ω)⊂hom⁡K(L)\hom_K(L,\Omega)\subset \hom_K(L)homK​(L,Ω)⊂homK​(L)

  1. 存在某个域扩张 Ω/L^\Omega/_LΩ/L​，其中 Ω\OmegaΩ 是代数封闭域，成立 hom⁡K(L,Ω)⊂hom⁡K(L)\hom_K(L,\Omega)\subset \hom_K(L)homK​(L,Ω)⊂homK​(L)
  2. 对任意不可约多项式 P∈K[X]P\in K[X]P∈K[X]，若 P(X)P(X)P(X) 在 LLL 中有根，则 PPP 在 L[X]L[X]L[X] 中分裂。
