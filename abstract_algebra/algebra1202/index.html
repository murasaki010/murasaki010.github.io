# 可分扩张

定义 4.2.3.1 KKK 是域，P∈K[X]P\in K[X]P∈K[X] 是 KKK - 系数多项式。给出对于不同对象的可分性的定义：

  2. L/K^L/_KL/K​ 是代数扩张，对于 x∈Lx\in Lx∈L，如果其在 KKK 中的极小多项式是可分的，就称 xxx 是 可分的；否则称之为 不可分的。

# 单扩张

定义： 给定域扩张 L/K^L/_KL/K​，如果存在 x∈Lx\in Lx∈L，使得 L=K(x)L=K(x)L=K(x)。称 L/K^L/_KL/K​ 是 单扩张，并称 xxx 是一个 本原元素。

L/K单扩张⟺L/K只有有限个中间域^L/_K \text{ 单扩张 }\iff {^L/_K}\text{ 只有有限个中间域}L/K​ 单扩张 ⟺L/K​ 只有有限个中间域

  1. K(a0,...,am)⊂MK(a_0,...,a_m)\subset MK(a0​,...,am​)⊂M

  3. [K(x):K(a0,...,am−1)]≤deg⁡(PM)=[L:M][K(x):K(a_0,...,a_{m-1})]\leq \deg(P_M)=[L:M][K(x):K(a0​,...,am−1​)]≤deg(PM​)=[L:M]
