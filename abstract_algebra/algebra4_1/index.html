# 环论基本概念

# 环

定义：称非空集合 AAA ，且 ∣A∣≥2|A|\geq 2∣A∣≥2，在其上定义的乘法和加法构成 环 (A,+,⋅)(A,+,\cdot \ )(A,+,⋅ )，即定义两个二元运算

A×A→A,(a1,a2)→a1+a2A\times A\to A,\quad (a_1,a_2)\to a_1+a_2A×A→A,(a1​,a2​)→a1​+a2​

和

A×A→A,(a1,a2)→a1⋅a2A\times A\to A,\quad (a_1,a_2)\to a_1\cdot a_2A×A→A,(a1​,a2​)→a1​⋅a2​

如果这两个映射满足

  1. (A,+)(A,+)(A,+) 是 Abel 群，0A0_A0A​ 是单位元；
  2. (A,⋅)(A,\cdot\ )(A,⋅ ) 是幺半群，1A≠0A1_A\neq 0_A1A​=0A​ 是单位元；
  3. 乘法 ⋅\cdot⋅ 对加法 +++ 满足左右分配律

称 AAA 为 交换环，如果 AAA 的乘法可交换。

称 B⊂AB\subset AB⊂A 是 AAA 的 子环，如果 (B,+)<(A,+)(B,+)<(A,+)(B,+)<(A,+)、1∈B1\in B1∈B 且 BBB 对乘法封闭。

称 AAA 为 可除环，如果 A×=A−{0}A^\times =A-\{0\}A×=A−{0}，其中 A×A^\timesA× 是 AAA 中的乘法可逆元之集，这是一个群。

称 AAA 为 域，如果 AAA 是可交换的可除环。

# 理想

# 定义

定义：AAA 是环，I⊂AI\subset AI⊂A 是 AAA 的加法子群。称 III 是 AAA 的 左（右）理想，如果

∀a∈A,x∈I,a⋅x∈I(x⋅a∈I)\forall a\in A,x\in I,\quad a\cdot x\in I\ \ (x\cdot a\in I)∀a∈A,x∈I,a⋅x∈I (x⋅a∈I)

显然对于真理想 III，1∉I1\notin I1∈/I

# 极大理想

# 定义

定义： AAA 是环，A≠m⊂AA\neq \mathfrak m\subset AA=m⊂A 是理想，称之为 极大理想，如果对理想 III

∀m⊂I⊂A⟹(I=m)∨(I=A)\forall\ \mathfrak m\subset I\subset A\implies (I=\mathfrak m)\lor (I=A)∀ m⊂I⊂A⟹(I=m)∨(I=A)

# 存在性

性质：AAA 是环， III 是理想，且 A≠IA\neq IA=I ，则存在极大理想 m\mathfrak mm，使得

I⊂m≠AI\subset \mathfrak m\neq AI⊂m=A

考虑包含 III 的理想在包含关系下的偏序集 J={J⊃I,J≠A}\mathcal J=\{J\supset I, J\neq A\}J={J⊃I,J=A}，考虑全序子集 S⊂JS\subset \mathcal JS⊂J，令

# 定义
