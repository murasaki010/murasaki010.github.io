# 可定向曲面

Definition. 设 Σ=⋃α∈ΛΣα\Sigma=\bigcup_{\alpha\in \Lambda}\Sigma_\alphaΣ=⋃α∈Λ​Σα​ 是 R3\mathbb R^3R3 中的曲面，其中 Σα\Sigma_\alphaΣα​ 为曲面片。对 α∈Λ\alpha\in \Lambdaα∈Λ，称曲面片 Σα\Sigma_\alphaΣα​ 的单位法向量场 nα\pmb n_\alphannα​ 为 Σα\Sigma_\alphaΣα​ 的定向。若对任意 α,β∈Λ\alpha,\beta\in \Lambdaα,β∈Λ，在 Σα∩Σβ≠∅\Sigma_\alpha\cap \Sigma_\beta\ne \varnothingΣα​∩Σβ​=∅ 时，有

nα=nβ\pmb n_\alpha=\pmb n_\betannα​=nnβ​

就称曲面 Σ\SigmaΣ 是可定向的，并称 {nα}α∈Λ\{\pmb n_\alpha\}_{\alpha\in \Lambda}{nnα​}α∈Λ​ 为 Σ\SigmaΣ 的定向。这时 n(P)=nα(P)\pmb n(P)=\pmb n_\alpha(P)nn(P)=nnα​(P)，当 P∈ΣαP\in \Sigma_\alphaP∈Σα​ 时，定义良好，是 Σ\SigmaΣ 的一个光滑单位法向量场。

# 紧致曲面

Definition. 曲面 Σ\SigmaΣ 的边界定义为所有边界点的集合，边界点 PPP 定义为存在曲面片 Σα\Sigma_\alphaΣα​，使得 P∈ΣαP\in \Sigma_\alphaP∈Σα​，且对应的参数区域 DαD_\alphaDα​ 的点 rα−1(P)\pmb r_\alpha^{-1}(P)rrα−1​(P) 是 DαD_\alphaDα​ 的边界点。

Remark. 边界点的定义是与曲面片有关的，因此边界的定义也与曲面片的选取有关。

Definition. 设 Σ\SigmaΣ 称为紧致曲面，若 Σ\SigmaΣ 是 R3\mathbb R^3R3 中无边的有界闭集。即曲面可以由没有边界点的曲面片拼接而成。

Example. 球面、环面是紧致曲面，平面、圆柱面不是紧致曲面。

# 三角剖分

Definition. Σ\SigmaΣ 上的一个三角形 TTT 是指

  1. TTT 是由 Σ\SigmaΣ 上三条光滑曲线围成的单连通区域；
  2. TTT 落在某个曲面片 Σα\Sigma_\alphaΣα​ 上。

曲面 Σ\SigmaΣ 的三角剖分是指 Σ\SigmaΣ 上的一族三角形 {Tλ}\{T_\lambda\}{Tλ​}，满足

  1. 对 Σ\SigmaΣ 的任意一点 PPP，PPP 必落在某个三角形上，且 PPP 不落在三角形的边界上时，PPP 属于唯一一个三角形；
  2. 当 PPP 属于某个三角形 T1T_1T1​ 的边，且不是三角形的顶点时，则或者 PPP 不再属于其他三角形的边，或者 PPP 也属于另外唯一一个三角形 T2T_2T2​ 的边，且 PPP 属于 T1∪T2T_1\cup T_2T1​∪T2​ 的内部；
  3. PPP 属于某个三角形的顶点时，则以 PPP 为顶点的三角形至多只有有限个，且两个三角形至多有一个公共边。
